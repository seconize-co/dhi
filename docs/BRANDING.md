## Pronunciation

**Dhi** - Pronounced "dhee" (rhymes with "free")

The 'dh' is a voiced aspirated dental stop in Sanskrit, but English speakers can pronounce it as a regular 'd' sound.

## Etymology

From the Sanskrit root धी (dhī):
- Found in Rigveda (ancient Hindu texts)
- Associated with Saraswati (goddess of knowledge)
- Root of words like "dhyana" (meditation)
- Represents the highest form of perception

## Why "Dhi"?

1. **Meaningful**: Represents intelligent perception - what security requires
2. **Short**: 3 letters, easy to type, memorable
3. **Global**: Sanskrit is neutral, not tied to any modern nation
4. **Unique**: No other security tool uses this name
5. **Pronounceable**: Works in most languages
